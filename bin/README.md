# Compiled Executables

This directory contains the compiled executables for chkiso.

## Contents

- `chkiso.exe` - The main Windows executable (compiled from chkiso.ps1)
- `chkiso.exe.sha` - SHA256 checksum for the executable

## Building

The executable is automatically built by GitHub Actions. To build manually, see the main README.md.

## Note

Executables in this directory are ignored by git (see .gitignore). Download official builds from the [Releases](https://github.com/pappasjfed/chkiso/releases) page.
